# 🍽️ Mealify – Recipe Finder App

Mealify is a **React Native mobile application** that allows users to search and explore delicious recipes using the **TheMealDB API**.

Users can search meals, view recipe details, and save their favorite dishes in a clean and responsive interface.

---

## 📱 Features

### Core Features
- 🔍 Search for meals
- 🍲 Display recipe cards (image, name, category)
- 📖 Recipe details screen
- ⏳ Loading skeleton while fetching data
- ❌ Error handling

### Bonus Features
- ❤️ Save favorite recipes
- 🌙 Theme toggle (Light / Dark mode)
- ✨ Smooth screen transitions

---

## 🛠 Tech Stack

- **React Native CLI**
- **React Navigation**
- **Redux Toolkit**
- **Redux Saga**
- **AsyncStorage**
- **TypeScript**
- **Axios**
